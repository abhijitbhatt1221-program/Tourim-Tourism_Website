# 🌍 Turism - Your Gateway to Exploration

A modern, responsive landing page for a travel agency, featuring tour packages, an about section, and an interactive contact form.

## 🚀 Features

* **Responsive Layout:** Optimized for desktop and mobile using CSS Flexbox.
* **Hero Section:** High-impact introduction with a call-to-action.
* **Tour Catalog:** Organized cards showcasing European, Asian, and African destinations.
* **Interactive Form:** JavaScript-validated contact form with submission feedback.
* **Clean Navigation:** Smooth internal linking and external page support.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, Flexbox, and hover effects.
* **JavaScript:** DOM manipulation and event handling.

---

## 📂 Project Structure

| File | Description |
| :--- | :--- |
| `index.html` | The main entry point and landing page. |
| `tour details.html` | Secondary page for specific tour information. |
